## What's Deploying

- **🛡️ Open-Source Tooling:** Lightweight scripts for security testing and automation.
- **📚 Playbooks & Guides:** Structured methodologies and security cheatsheets.
- **🔍 Vulnerability Research:** Proof-of-concepts, write-ups, and security analysis.
- **🛠️ DevSecOps Utilities:** Integrations to shift security left in the SDLC.

## Philosophy

At **SimplySec**, we believe that complexity is the enemy of security.

Some projects begin as internal scripts.  
Some turn into useful open-source tools.  
Some are documentation to make complex concepts simple.

**Analyze fast. Harden often. Secure always.**
